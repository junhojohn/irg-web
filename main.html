IRG(Integration Request document Generator)

IRG(Integration Request document Generator)는 SVN으로부터 가져온 Base 코드 파일 리스트와 Update 코드 파일 리스트를 조합하여 메인 프로젝트의 IR 문서 또는 파생 프로젝트의 IR문서를 자동으로 생성하는 기능을 제공하는 도구입니다.

IRGenerator가 제공하는 특징은 아래와 같습니다.

  • 설정파일과 연계한 프로젝트 정보 입력 및 정보 관리가 가능합니다.
  • 프로젝트 SVN 정보(파일 명, 리비전 번호, 파일 위치)를 자동 로드 후 출력합니다.
  • 메인 프로젝트와 파생 프로젝트 각각 용도에 맞게 IR 문서 출력이 가능합니다.
